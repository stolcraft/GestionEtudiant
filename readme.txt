0- Dans le dossier avec le src et le POM,
lancer une commande cmd et écrivez la phrase
"mvn clean package"

1- Deplacer le fichier DemoJpaServlet.war dans le dossier target et mettez le 
dans votre serveur.

2- Crée une base de donnée demojpa vide dans mysql

2- (Pour tomcat )Executer startup.bat situé dans le dossier bin de tomcat

4- Dans l'url du navigateur internet, entrer la commande suivante : http://localhost:8080/DemoJpaServlet/

5- Les identifiants de connection sont :
	Login : root
	Mdp : root

6- Cela affiche le tableau de bord qui permet :
	- De voir l'ensemble des étudiants
	- D'ajouter un étudiant
	- De supprimer un étudiant
	- De modifier un étudiant

7-Vous pouvez voir l'ensemble des informations dans la console tomcat
